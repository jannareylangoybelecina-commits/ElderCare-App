# ElderCare Screenshots Guide

Use this checklist to keep screenshots organized for demos, reports, and repository presentation.

## Screenshot Naming Convention

Use consistent names:

- `01-role-selection.png`
- `02-login.png`
- `03-forgot-password.png`
- `04-elderly-home.png`
- `05-reminders.png`
- `06-medication-tracker-done-missed.png`
- `07-set-reminder.png`
- `08-health-reading-entry.png`
- `09-set-appointment.png`
- `10-notifications.png`
- `11-reading-results.png`
- `12-missed-medications.png`
- `13-caregiver-dashboard.png`
- `14-settings.png`

## Recommended Folder

Store screenshots in:

- `docs/screenshots/`

## Capture Checklist (What to Show)

- Role selection screen
- Login and register screens
- Forgot password screen (instruction clarity)
- Elderly dashboard tabs:
  - Home
  - Set Reminder
  - Notification
  - Health History
- Medication status examples:
  - `DONE` (green)
  - `MISSED` (red)
- Caregiver dashboard
- Settings and support pages

## Quality Guidelines

- Use portrait orientation
- Keep status bar visible for realistic previews
- Capture before/after when showing improvements
- Avoid personal email/real private data in screenshots
